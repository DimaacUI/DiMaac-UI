# Mara Vance · Portfolio Landing

Designer portfolio landing — WebGL hero shader, GSAP preloader, scroll-linked work grid, custom cursor.

## Files

```
portfolio-landing/
├── index.html
├── styles.css
├── main.js
└── vendor/            # Three.js + GSAP + ScrollTrigger
```

## Run locally

```bash
npx serve .
```

## Customize

1. Update name, bio, and work items in `index.html`
2. Edit WebGL shader uniforms in `main.js` if desired
3. Typography and colors in `styles.css`

Work cards use CSS art — replace with `<img>` tags for real project shots.
