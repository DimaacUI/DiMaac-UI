# Minimal · Ember

**Mara Costa** — warm cream portfolio with floating mesh gradient blob (CSS + WebGL).

## Files

- `index.html` — structure
- `styles.css` — warm palette, badge gradient border, layout
- `main.js` — ember WebGL noise field

## Run locally

```bash
npx serve .
```

## Customize

- Ember colors: `--e1` … `--e4` in `styles.css`
- Tag dot classes `.t1`–`.t4` for project labels
