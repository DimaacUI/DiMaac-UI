# Minimal · Quiet

**Arav Mehta** — one-column serif portfolio. Calm typography, no dependencies.

## Files

- `index.html` — structure
- `styles.css` — layout & motion
- `main.js` — live clock

## Run locally

```bash
npx serve .
# or: python3 -m http.server 8080
```

Open `http://localhost:8080`

## Customize

1. Replace name, bio, projects, and experience in `index.html`
2. Tweak colors in `:root` inside `styles.css`
3. Update Google Font in `index.html` if needed

No build step. Deploy any static host (Netlify, Vercel static, GitHub Pages).
